# 🎵 Jetpack Compose Music Player

An open-source **Music Player** built with **Jetpack Compose** that supports all audio formats using **Android Media Player**. It provides a **beautiful and user-friendly UI/UX**, supporting **light & dark mode**, and allows users to play, pause, resume, and navigate through their songs easily.

## ✨ Features

- 🎵 **Supports all audio formats** (MP3, WAV, AAC, etc.)
- 🌗 **Light & Dark Mode**
- 🎛️ **Playback Controls** (Play, Pause, Resume, Forward, Backward)
- 📂 **Multiple Lists**
  - Albums List
  - Audio Folders List
  - Device All Songs List
  - Artist List
- 🎨 **Modern UI/UX** with a stylish player screen
- 🏗️ **Built with Jetpack Compose** for a declarative UI

## 🚀 Installation

1. Clone the repository:
   ```sh
   gh repo clone UmairOye/MusicPlayer-JetpackCompose
   ```
2. Open the project in **Android Studio**
3. Connect a physical device or use an emulator
4. Run the app and enjoy! 🎶

## 🛠 Tech Stack

- **Language:** Kotlin
- **Framework:** Jetpack Compose
- **Media Handling:** Android MediaPlayer API
- **Architecture:** MVVM + Clean Architecture
